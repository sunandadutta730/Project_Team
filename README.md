# 🚀 Project Team — About Us

A modern, visually stunning **"About Us"** web page built for the JISSP college project team. Designed to showcase team members, their roles, and the collective mission — with a futuristic, premium dark-themed UI.

![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black)

---

## ✨ Features

- **🌌 Futuristic Dark Theme** — Sleek dark UI with animated glowing orbs, grid overlay, and glassmorphism effects
- **👥 Team Member Cards** — Interactive cards with avatar, name, role, and "View Profile" links
- **📄 Individual Profile Pages** — Dedicated page for each member with bio, skills, education & contact details
- **🖼️ Profile Photos** — Circular bordered images with a neon glow effect
- **🎯 Mission Section** — Highlights the team's core goals with icon-driven cards
- **📊 Animated Stats Bar** — Counters that animate on scroll (team size, disciplines, etc.)
- **🎬 Scroll Reveal Animations** — Elements smoothly fade in as you scroll down the page
- **📱 Fully Responsive** — Looks great on desktop, tablet, and mobile

## 🚀 Getting Started

### Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sunandadutta730/Project_Team.git
   ```

2. **Navigate to the project folder:**
   ```bash
   cd Project_Team
   ```

3. **Open in browser** — Simply open `index.html` in any modern browser, or start a local server:
   ```bash
   python -m http.server 8000
   ```
   Then visit `http://localhost:8000`
